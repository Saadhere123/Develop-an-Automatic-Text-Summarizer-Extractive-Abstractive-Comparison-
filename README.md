                                          AI Text Summarizer (Extractive vs Abstractive)

Overview
This project implements an automatic text summarization system using two approaches:
- Extractive Summarization (TF-IDF based)
- Abstractive Summarization (Transformer models)

Technologies Used
- Python
- NLTK
- Scikit-learn
- Hugging Face Transformers
eatures
- Extractive summary generation
- Abstractive summary using T5 model
- CLI-based interface
- Custom text input
- Sample dataset support
